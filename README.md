### GitStats
Git blame visualizer to flex on your friends

Requires Qt creator to run :+1:

---
How to use:

  Actions:

  Select->Folder: Selects folder/git repo
  Select->Reload Folder: Checks the current directory for changes

  Request->Pull: Requests a pull from git

Pretty easy right?

Navigate throught the tabs to find:

- Status
- Overall blame (who did the most)
- File blame (select a file and see who did the most)
